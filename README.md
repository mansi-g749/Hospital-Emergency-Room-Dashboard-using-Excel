# 🏥 Hospital Emergency Room Dashboard (Excel Project)

Excel-based Hospital Emergency Room Dashboard analyzing patient flow, wait times, satisfaction, and referrals using interactive pivot tables and slicers.

## 📌 Project Overview
This project is an interactive Hospital Emergency Room Dashboard created using Microsoft Excel. It provides insights into patient visits, department performance, and emergency trends to help in better decision-making.

The dashboard is designed to analyze hospital data efficiently and present it in a clear and visual format.

---

## 🎯 Objectives
- Analyze patient inflow in emergency rooms
- Track department-wise performance
- Identify peak hours and busy days
- Improve hospital resource management

---

## 🛠️ Tools & Technologies Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning & Formatting

---

## 📊 Key Features
- Interactive dashboard with filters (year, department, etc.)
- Patient count analysis
- Department-wise comparison
- Time-based trends visualization
- Clean and user-friendly layout

---

## 📁 Dataset Details
The dataset includes:
- Patient details
- Department information
- Date and time of visit
- Emergency case categories

---

## 🚀 How to Use
1. Download the Excel file
2. Open in Microsoft Excel
3. Use slicers to filter data
4. Explore insights from charts and tables

---

## 📈 Insights Gained
- Identified peak emergency hours
- Found most active departments
- Observed patient trends over time
- Helped in better resource planning
